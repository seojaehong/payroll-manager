## 이미지 가이드라인

### 일반 원칙
- 해상도: 72dpi (웹용)
- 포맷: PNG (UI 캡처) 또는 JPG (사진)
- 파일명: `part-XX-screenshot-YY.png` 형식

### 개인정보 처리
- 주민등록번호: `950101-1******` 형태로 마스킹
- 이름: `김**`, `이**` 형태로 마스킹
- 사업자번호: 그대로 사용 가능 (공개 정보)
- 연락처: `010-****-1234` 형태로 마스킹

### 스타일 권장
- 다크 모드 화면 캡처 권장 (블로그 일관성)
- 불필요한 브라우저 UI는 크롭
- 중요 부분에 빨간색 박스나 화살표 추가 가능
